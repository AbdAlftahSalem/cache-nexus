# Contributing to Smart Cache

Thanks for your interest in contributing! Here's how to get started.

## Development Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/AbdAlftahSalem/cache-nexus.git
   cd cache-nexus
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the example app:
   ```bash
   cd example
   flutter run
   ```

## Before Submitting a PR

Run these commands and make sure they pass:

```bash
# Static analysis
flutter analyze

# Tests
flutter test
```

## Pull Request Guidelines

- Create a feature branch from `main` (`git checkout -b feature/your-feature`)
- Keep PRs focused on a single change
- Add tests for new functionality
- Update documentation if needed
- Follow existing code style (the linter enforces it)

## Reporting Issues

Use the [GitHub issue tracker](https://github.com/AbdAlftahSalem/cache-nexus/issues) to report bugs or request features.
